This website helps the user to calculate the age of person if date of  birth is given. The website contains four tabs:
1. Home
The home page will introduce the author and a video about his college northwest  Missouri state university.
2. Age Calculator
This page will help the user to calculate the age of a given person if his  date of birth is inputted.
3. Contact
This page provides a way to contact the author through the mail or he can use  the navigates system.
4. Daily Inspiration
Here the author describes about the person and his famous quotes which  inspired him a lot.
